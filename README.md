# SIO Backend

## 🚀 SIO Backend
API del Sistema Integrado de Operadores (SIO) construida con **Node.js**, **Express** y **PostgreSQL**.

---

## 📦 Requisitos
- Node.js 18+
- npm
- PostgreSQL 14+
- Archivo `.env` configurado

---

## 🔧 Instalación

Instala todas las dependencias necesarias ejecutando:

```bash
npm install
```

---

## 📦 Iniciar el servidor

Inicia el servidor ejecutando:

```bash
npm run dev
```
